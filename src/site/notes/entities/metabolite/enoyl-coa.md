<h1>Enoyl-CoA</h1>
<h2>Overview</h2>
<p>Enoyl-CoA (trans-Δ2-enoyl-CoA) is the first intermediate in the <a href="/garden/michal.k7w@gmail.com/pathway/beta-oxidation/" class="internal-link">Beta-Oxidation</a> spiral, formed when <a href="/garden/michal.k7w@gmail.com/protein/acyl-coa-dehydrogenase/" class="internal-link">Acyl-CoA Dehydrogenase</a> oxidizes the fatty acyl-CoA chain, producing <a href="/garden/michal.k7w@gmail.com/metabolite/fadh2/" class="internal-link">FADH2</a>.</p>
<h2>Mechanism of Action</h2>
<ul>
<li><strong>Formation</strong>: Acyl-CoA dehydrogenase oxidizes fatty acyl-CoA → trans-Δ2-enoyl-CoA + <a href="/garden/michal.k7w@gmail.com/metabolite/fadh2/" class="internal-link">FADH2</a></li>
<li><strong>FADH2 disposal</strong>: Electrons from FADH2 enter the <a href="/garden/michal.k7w@gmail.com/pathway/electron-transport-chain/" class="internal-link">Electron Transport Chain</a> via electron-transferring flavoprotein (ETF) → ETF-ubiquinone oxidoreductase → CoQ</li>
<li><strong>Hydration</strong>: Enoyl-CoA hydratase adds H2O across the trans double bond → <a href="/garden/michal.k7w@gmail.com/metabolite/3-hydroxyacyl-coa/" class="internal-link">3-Hydroxyacyl-CoA</a></li>
</ul>
