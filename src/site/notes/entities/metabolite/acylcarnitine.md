<h1>Acylcarnitine</h1>
<h2>Overview</h2>
<p>Acylcarnitine is the transport form of activated fatty acids, essential for crossing the inner mitochondrial membrane. The carnitine shuttle system (<a href="/garden/michal.k7w@gmail.com/protein/cpt-i/" class="internal-link">CPT-I</a> → translocase → <a href="/garden/michal.k7w@gmail.com/protein/cpt-ii/" class="internal-link">CPT-II</a>) is the rate-limiting gateway for <a href="/garden/michal.k7w@gmail.com/pathway/beta-oxidation/" class="internal-link">Beta-Oxidation</a> of long-chain fatty acids.</p>
<h2>Mechanism of Action</h2>
<ul>
<li><strong>Formation</strong>: <a href="/garden/michal.k7w@gmail.com/protein/cpt-i/" class="internal-link">CPT-I</a> catalyzes fatty acyl-CoA + carnitine → acylcarnitine + CoA</li>
<li><strong>Transport</strong>: Carnitine-acylcarnitine translocase exchanges acylcarnitine (in) for carnitine (out)</li>
<li><strong>Reconversion</strong>: <a href="/garden/michal.k7w@gmail.com/protein/cpt-ii/" class="internal-link">CPT-II</a> catalyzes acylcarnitine + CoA → <a href="/garden/michal.k7w@gmail.com/metabolite/fatty-acyl-coa/" class="internal-link">Fatty Acyl-CoA</a> + carnitine (in matrix)</li>
<li><strong>Rate control</strong>: <a href="/garden/michal.k7w@gmail.com/protein/cpt-i/" class="internal-link">CPT-I</a> is inhibited by malonyl-CoA (fed state), preventing futile cycling</li>
</ul>
<h2>Clinical Significance</h2>
<ul>
<li><strong>Acylcarnitine profiling</strong> is a key diagnostic test for fatty acid oxidation disorders (FAODs)</li>
<li>Elevated acylcarnitine species in blood indicate specific enzyme deficiencies (e.g., MCAD deficiency → elevated C8-acylcarnitine)</li>
<li><a href="/garden/michal.k7w@gmail.com/supplement/l-carnitine/" class="internal-link">L-Carnitine</a> supplementation can restore acylcarnitine transport capacity</li>
</ul>
