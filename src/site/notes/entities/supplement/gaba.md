<h1>GABA</h1>
<h2>Overview</h2>
<p>GABA (gamma-aminobutyric acid) is the brain's primary inhibitory neurotransmitter, present in approximately 40% of all synapses. It provides essential &quot;braking&quot; action to counterbalance excitatory neurotransmission (primarily glutamate). GABA dysfunction is implicated in anxiety disorders, insomnia, epilepsy, and many neurological conditions.</p>
<h2>Synthesis</h2>
<pre><code>GLUTAMATE
    ↓ (Glutamic Acid Decarboxylase - GAD)
    ↓ (Requires Vitamin B6/P5P as cofactor)
GABA
</code></pre>
<p>Key enzyme: GAD (Glutamic Acid Decarboxylase)</p>
<ul>
<li>GAD65: Primarily in nerve terminals</li>
<li>GAD67: Throughout neuron</li>
</ul>
<h2>Receptor Types</h2>
<h3>GABA-A Receptors</h3>
<ul>
<li>Ionotropic (ligand-gated chloride channel)</li>
<li>Fast inhibition (milliseconds)</li>
<li>Target of benzodiazepines, barbiturates, alcohol</li>
<li>Multiple binding sites for modulation</li>
</ul>
<h3>GABA-B Receptors</h3>
<ul>
<li>Metabotropic (G-protein coupled)</li>
<li>Slow, prolonged inhibition</li>
<li>Target of baclofen</li>
<li>Affects calcium and potassium channels</li>
</ul>
<h2>Clinical Significance</h2>
<h3>Low GABA</h3>
<ul>
<li>Anxiety</li>
<li>Insomnia</li>
<li>Restlessness</li>
<li>Panic attacks</li>
<li>Seizures</li>
<li>Chronic stress</li>
<li>Racing thoughts</li>
</ul>
<h3>GABA-Ergic Medications</h3>
<ul>
<li>Benzodiazepines (Valium, Xanax)</li>
<li>Gabapentinoids (Gabapentin, Pregabalin)</li>
<li>Barbiturates</li>
<li>Z-drugs (Ambien, Lunesta)</li>
<li>Alcohol (initial effect)</li>
</ul>
<h2>Supplements That Affect GABA</h2>
<h3>Direct GABA Supplementation</h3>
<ul>
<li><strong>GABA</strong>: 250-750 mg (controversial BBB penetration)</li>
<li><strong>Pharma-GABA</strong>: Fermented form, may cross BBB better</li>
<li><strong>Picamilon</strong>: GABA + niacin (better BBB penetration)</li>
</ul>
<h3>GABA Precursors</h3>
<ul>
<li><strong>L-Glutamine</strong>: Converts to glutamate, then GABA</li>
<li><strong>L-Theanine</strong>: Increases GABA, promotes alpha waves</li>
</ul>
<h3>GABA Modulators</h3>
<ul>
<li><strong>L-Theanine</strong>: Increases GABA, reduces anxiety (100-400 mg)</li>
<li><strong>Magnesium</strong>: GABA-A receptor positive modulator</li>
<li><strong>Taurine</strong>: GABA-A agonist, calming</li>
<li><strong>Valerian</strong>: GABA-A modulation, sleep support</li>
<li><strong>Passionflower</strong>: GABA-A modulation</li>
<li><strong>Lemon Balm</strong>: GABA transaminase inhibition</li>
<li><strong>Kava</strong>: GABA-A modulation (potent, use cautiously)</li>
</ul>
<h3>Cofactors for GABA Synthesis</h3>
<ul>
<li><strong>Vitamin B6 (P5P)</strong>: Essential cofactor for GAD</li>
<li><strong>Zinc</strong>: Modulates GABA receptors</li>
<li><strong>Magnesium</strong>: Supports GABAergic function</li>
</ul>
<h2>GABA vs Glutamate Balance</h2>
<p>The brain maintains critical balance:</p>
<ul>
<li><strong>Glutamate</strong>: Excitatory (accelerator)</li>
<li><strong>GABA</strong>: Inhibitory (brake)</li>
</ul>
<p>Imbalance causes:</p>
<ul>
<li>Excess glutamate → Excitotoxicity, anxiety, seizures</li>
<li>Excess GABA → Sedation, cognitive impairment</li>
</ul>
<p>Supporting balance:</p>
<ul>
<li>Adequate B6 for GABA synthesis</li>
<li>Magnesium (reduces glutamate receptors)</li>
<li>Taurine (modulates both)</li>
<li>Glycine (inhibitory support)</li>
</ul>
<h2>Lifestyle Factors</h2>
<h3>GABA-Boosting</h3>
<ul>
<li>Yoga (increases GABA 27% in one study)</li>
<li>Meditation</li>
<li>Regular exercise</li>
<li>Deep breathing</li>
<li>Adequate sleep</li>
<li>Stress reduction</li>
</ul>
<h3>GABA-Depleting</h3>
<ul>
<li>Chronic stress</li>
<li>Caffeine excess</li>
<li>Alcohol withdrawal</li>
<li>Poor sleep</li>
<li>Inflammatory states</li>
</ul>
