<h1>Alpha-Synuclein Aggregation</h1>
<h2>Overview</h2>
<p>Alpha-synuclein aggregation is the central pathological process underlying Parkinson's disease and related synucleinopathies. The normally soluble 140-amino acid protein misfolds and self-assembles into toxic oligomers and insoluble fibrils, which accumulate in neurons as Lewy bodies and Lewy neurites. This process leads to synaptic dysfunction, neuroinflammation, and neuronal death.</p>
<h2>Normal Alpha-Synuclein Function</h2>
<ul>
<li>Concentrated at presynaptic terminals</li>
<li>Regulates synaptic vesicle trafficking</li>
<li>Modulates dopamine release</li>
<li>Interacts with lipid membranes</li>
<li>May function as a chaperone</li>
</ul>
<h2>Aggregation Cascade</h2>
<h3>1. Monomeric State</h3>
<ul>
<li>Native, unfolded protein</li>
<li>Membrane-bound: alpha-helical</li>
<li>Cytoplasmic: intrinsically disordered</li>
</ul>
<h3>2. Oligomer Formation</h3>
<ul>
<li>Misfolded monomers associate</li>
<li><strong>Most toxic species</strong></li>
<li>Membrane permeabilization</li>
<li>Mitochondrial dysfunction</li>
</ul>
<h3>3. Protofibril Stage</h3>
<ul>
<li>Ordered beta-sheet structures</li>
<li>Intermediate aggregates</li>
<li>Seeding capability</li>
</ul>
<h3>4. Mature Fibrils</h3>
<ul>
<li>Insoluble amyloid structures</li>
<li>Cross-beta sheet architecture</li>
<li>May be less acutely toxic than oligomers</li>
</ul>
<h3>5. Lewy Bodies</h3>
<ul>
<li>Dense cytoplasmic inclusions</li>
<li>Surrounded by organelles and cytoskeletal elements</li>
<li>Diagnostic hallmark of PD</li>
</ul>
<h2>Triggers and Risk Factors</h2>
<h3>Genetic</h3>
<ul>
<li><strong>SNCA gene</strong>: Duplications, triplications, point mutations</li>
<li><strong>GBA1 mutations</strong>: Impaired lysosomal function</li>
<li><strong>LRRK2 mutations</strong>: Phosphorylation abnormalities</li>
</ul>
<h3>Environmental</h3>
<ul>
<li>Pesticides (rotenone, paraquat)</li>
<li>Heavy metals</li>
<li>Oxidative stress</li>
</ul>
<h3>Cellular Stress</h3>
<ul>
<li>Mitochondrial dysfunction</li>
<li>Proteasome impairment</li>
<li>ER stress</li>
<li>Aging</li>
</ul>
<h2>Prion-Like Spreading</h2>
<h3>Cell-to-Cell Transmission</h3>
<ol>
<li>Misfolded alpha-synuclein released from neurons</li>
<li>Uptake by neighboring cells</li>
<li>Templated seeding of native protein</li>
<li>Spreading along neuronal networks</li>
</ol>
<h3>Braak Staging</h3>
<ul>
<li>Pathology starts in olfactory bulb and enteric nervous system</li>
<li>Spreads to brainstem (Stage 1-2)</li>
<li>Reaches substantia nigra (Stage 3)</li>
<li>Progresses to cortex (Stage 4-6)</li>
</ul>
<h2>Therapeutic Targets</h2>
<h3>Aggregation Inhibition</h3>
<ul>
<li>Small molecule aggregation inhibitors</li>
<li>Molecular chaperones</li>
<li>Heat shock proteins</li>
</ul>
<h3>Clearance Enhancement</h3>
<ul>
<li>Autophagy activation</li>
<li>Lysosomal function restoration</li>
<li>Ubiquitin-proteasome enhancement</li>
</ul>
<h3>Immunotherapy</h3>
<ul>
<li>Active vaccines (affitope)</li>
<li>Passive immunization (anti-α-syn antibodies)</li>
<li>Prasinezumab, cinpanemab in trials</li>
</ul>
<h3>Gene Therapy</h3>
<ul>
<li>Antisense oligonucleotides</li>
<li>siRNA approaches</li>
<li>SNCA reduction strategies</li>
</ul>
